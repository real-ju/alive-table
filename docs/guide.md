# 指南

## 项目环境要求

- Vue+TypeScript（墙裂推荐使用[vue3-boot](https://github.com/real-ju/vue3-boot)搭建项目）
- less 或 sass
- ant-design-vue@3.x

## 安装

Github 仓库地址：[https://github.com/real-ju/alive-table](https://github.com/real-ju/alive-table)

clone 本仓库到本地，复制组件源文件到项目中。
